# 测试一下
		现在我要测试docs这本书
		打开docs文件夹，
		右键Git Bash Here
		输入
```java
docsify serve
```
		后，它不出意外会返回
```java
Listening at http://localhost:3000
```
		这时打开浏览器，访问http://localhost:3000可以看到预览的效果
		满意了，我们在Bash界面按ctrl+c终止测试。
		更多教程请去官网
* [https://docsify.js.org/#/quickstart](https://docsify.js.org/#/quickstart)
